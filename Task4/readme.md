# Shell Scripting Internship Task – Task 4

## Overview

This project was completed as part of an internship assignment to demonstrate Linux shell scripting fundamentals, automation techniques, and best scripting practices.

All required tasks are implemented inside a single script file named `script-4.sh`.

The script demonstrates:

- System information retrieval
- Use of variables
- Conditional statements
- Loop constructs
- Log backup and cleanup automation
- Output redirection
- Proper comments and documentation
- Execution permission handling

---

## How to Run the Script

### Step 1 — Grant Execute Permission

```
chmod +x script-4.sh
ls -l script-4.sh

```

### Step 2 — Grant Execute Permission
```
./script-4.sh

```

### Step 3 — Redirect Output to File
```
./script-4.sh > output.sh
cat output.sh

```
